GitHub Actions は、自動テストを実行します。これらは手動でのレビューに代替されるものではありませんが、共同作業に集中し、より高い価値を生み出すことに役立ちます。

GitHub Actions には通常...

- プロジェクト構築
- 自動化テスト
- コードフォーマットの整合性を確認するためのリンター
